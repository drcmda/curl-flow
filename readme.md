# Curl Flow

[Demo](https://curl-flow.vercel.app/)

This experiment is based on the following sources.

- Edan Kwan's [The Spirit](https://github.com/edankwan/The-Spirit)
- Bruno Simon's [Three.js Journey](https://threejs-journey.com/)
- [webgl gpgpu birds](https://threejs.org/examples/#webgl_gpgpu_birds)

## Setup

Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
